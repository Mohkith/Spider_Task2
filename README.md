# Spider_Task2

# Level 1

# Usage:
```sh
  ./Basic_Recon.py [FLAGS]
```
```yaml
Flags:
INPUT:
  -d  domain
  -nd out of scope domains
  -I ip_address
OUTPUT:
  -o output file
HELP:
  -h help
```

# Required Libraries
- argparse
- json
- requests
- sublist3r
- dns.resolver
- whois
- subprocess
- geoip2.database

Installing the Libraries:
``` sh
pip install <module_name>
```
